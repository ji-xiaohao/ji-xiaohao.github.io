机器学习

本版块展示了我在以下方向的部分实践成果：

·Python 数据分析

聚焦上市公司财务数据，运用 Python 进行数据清洗、指标构建与可视化分析，深入挖掘企业在财务健康状况、资本结构以及经营风险等方面的动态表现，为财务洞察与决策支持提供数据驱动依据。

·机器学习应用研究

基于 A 股市场公司数据，结合 XGBoost、GBDT、Lasso 等机器学习模型，系统研究企业成本结构、股权特征、审计意见识别及估值差异等实际问题，注重模型在真实业务场景中的可解释性与应用价值。

 

1.Python分析

Python——普利制药财务舞弊分析

本项目基于叶钦华等（2022）发表于《会计研究》的论文《财务舞弊识别框架构建——基于会计信息系统论及大数据视角》，借助 Python 对普利制药的财务数据进行系统性分析。通过构建盈利能力、现金流异常、关联交易等关键指标，尝试从数据中捕捉潜在的风险信号，综合评估该企业的财务健康状况与舞弊风险。

📄相关分析报告详见：Python_普利制药财务舞弊分析.html

Python——小米数据分析

本项目运用 Python 对小米集团的财务报表数据进行全面分析，并通过可视化手段动态呈现其关键财务指标的演变趋势与当前状态。研究重点包括资本结构、融资策略及财务风险水平等方面，旨在全面评估企业的财务表现与潜在风险。

📄相关分析报告详见：Python_小米数据分析.html

Python——多市场日收益交互看板

本项目旨在展示运用 Python 高级可视化库 Plotly 进行金融数据分析的专业能力。项目基于 CSMAR 数据库提供的“日市场回报率”数据，对多个主要市场（包括上证A股、深证A股、北证A股、创业板、科创板等）的日收益率展开了系统性与交互式的可视化探索。

📄相关分析报告详见：Python_多市场日收益交互看板.html

2.机器学习项目

机器学习——成本管理模型

基于2018年至2024年沪深京A股上市公司的年度财务数据，本文对企业成本与费用结构展开建模分析。在线性回归与梯度提升决策树（GBDT）的对比实验中，重点探索不同模型在拟合企业成本行为中的表现差异，初步评估各类算法在解释费用结构变化方面的适用性。

📄相关报告详见：机器学习_成本管理模型.html

机器学习——股权特征与企业市场价值

本项目借助机器学习方法，系统研究A股上市公司股权结构特征及其对企业市场价值的影响。通过可视化呈现两权分离度、机构持股比例等核心指标的分布，并从行业（制造业/非制造业）与所有制（国企/非国企）等维度进行分组对比，深入揭示不同类别企业在股权结构与市场价值关联中的异同。

在建模过程中，采用梯度提升树（GBDT）和Lasso回归算法，有效捕捉股权特征与企业价值之间的复杂关系，提升模型解释与预测准确性。

📄相关报告详见：机器学习_股权特征与企业市场价值.html

机器学习——关键审计事项多标签分类

本研究致力于探索不同类别的关键审计事项下，企业基本面特征对审计判断的差异化影响。聚焦五类高频审计事项：收入确认、其他事项、应收账款、无形资产和存货。基于上市公司公开审计报告与财务数据，构建多标签分类任务，并引入包括公司规模、上市年限、账面市值比、销售增长率、杠杆率、盈利能力及亏损状态等多维度解释变量。

应用XGBoost模型分别对每一类审计事项进行建模，结合特征重要性与SHAP值分析，识别不同情境下具有预测力的关键变量，从而揭示审计师在收入确认、无形资产减值等不同事项中的关注差异。

📄相关报告详见：机器学习_关键审计事项多标签分类.html

机器学习——国有、民营的估值差异

本项目利用XGBoost模型分析国企与民营企业之间的估值差异及其驱动因素。以市净率为被解释变量，综合考察企业性质、资本支出、股权集中度、公司规模、资产结构、财务杠杆、盈利能力和公司年龄等多方面因素的影响，旨在揭示不同所有制企业在市场估值中的表现差异与内在机制。

📄相关报告详见：机器学习_国有、民营的估值差异.html
