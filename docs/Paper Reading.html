1. 关键审计事项的意义与影响

📄 《谁更关心关键审计事项 :管理层还是分析师》(审计研究, 2020)

  • 2024年9月
  • 作者：薛刚, 王储, 赵西卜
  • 核心问题：在信息披露高要求的背景下，关键审计事项作为审计报告改革的核心，旨在提升信息透明度。文章管 理层行为的角度考察披露关键审计事项对资本市场的影响，提出：关键审计事项究竟是作为增量信息改进分析师预测,还是作为监督压力抑制管理层迎合分析师预测的行为？
  • 主要发现：
    • 主要通过“监督效应”而非“增量信息效应”发挥作用
    • 具有局部性：仅在管理层薪酬激励较弱、权力受到制衡（如董事会独立性高）时显著，说明治理结构是审计信息发挥效力的前提
  • 我的思考：
    • 关键审计事项的披露让管理层不敢轻易操纵利润来迎合分析师的预期了。也就是部分公司可能会“美化”财报，让业绩看起来刚好达到或超过分析师预测，在迎合分析师的预测时，也提升自己公司的股价。但随着关键审计事项的披露，审计更透明、监督更严，管理层怕被查，就不敢这么做了。
    • 也就是透明度的提升不等于信息更容易被预测，反而可能让市场看到更多“真实但复杂”的信息，从短期来看会让预测变难。但长期看，市场会不再依赖“被修饰过的信息”从而被稳定预期，而是更真实地反映企业风险。
    • 但并不是所有公司都需要同等强度的监督。对于治理差、风险高的公司，加强审计披露确实能“补短板”；但对于治理良好的公司，过度披露可能只是增加成本。

📄 《关键审计事项披露与企业投资效率——基于文本分析的经验证据》(审计研究, 2022)

  • 2024年9月
  • 作者：王帆, 邹梦琪
  • 核心问题：关键审计事项究竟是作为“有用信息”去引导资本高效配置，还是仅仅流于形式、对实际经济后果影响有限？尤其关注披露的文本质量（如详细程度、精确性、可读性）是否真正影响了企业的投资效率
  • 主要发现：
    • 关键审计事项披露得越详细、表述越精确（如明确指出风险领域、会计科目和判断依据），企业投资效率越高。高质量的审计披露确实能提供有用信息。
    • 市场更关注“说了什么”，而不是“怎么说”（可读性和语调对投资效率影响不大）。
    • 如果公司有更多分析师关注、机构投资者持股多、管理层不盲目自信，那么详细精确的审计披露效果更好。
  • 我的思考： -关键审计事项写得是否通俗易懂或语气是积极还是谨慎，对投资效率影响不大。也就是说，大家更关心“你说了什么重要信息”，而不是“你说话好不好读”或“语气够不够委婉”
    • 审计师不仅要说明问题是什么，还要讲清楚怎么查的，而且“问题是什么”这部分写得越清楚、越准确，越有助于提升投资效率。

📄 《关键审计事项披露与审计费用》(审计研究, 2020)

  • 2024年10月
  • 作者：周中胜, 贺超, 邵蔚
  • 核心问题：注册会计师披露“关键审计事项”，企业为此是否支付了更高的审计费用、经历了更长的审计周期？
  • 主要发现：
    • 新审计报告实施后，上市公司平均审计费用和审计时间（从年报截止到发布报告的时间）都明显上升
    • 如果一家公司长期由同一家会计师事务所审计（任期长），或者对该事务所来说是“大客户”，那么即使披露了很多复杂的KAMs，审计费用的增长也会被“缓冲”，不好增长明显
  • 我的思考：
    • 信息透明是有成本的。更透明的审计报告能提升市场信任，但这份“透明”是由企业和审计师共同承担代价换来的——更高的费用、更长的等待时间。
    • 但是对监管者而言，推动披露改革时不能只看“效果”，还要评估“成本”。尤其对中小型企业，应避免过度披露造成负担不均，需要实现透明度、效率与成本之间的平衡。

📄 《在审计报告中沟通关键审计事项是否提高了盈余质量?》(中国软科学, 2019)

  • 2024年10月
  • 作者：李延喜, 赛骞, 孙文章
  • 核心问题：披露关键审计事项，是否能有效抑制上市公司进行财务操纵
  • 主要发现：
    • 企业在被要求披露KAMs后，通过会计科目调整来操纵利润的程度（应计盈余管理）显著下降
    • 大规模企业披露KAMs后，利润操纵下降更明显；而小公司的变化不显著——可能因为小公司本身治理弱、审计资源投入少，或者监管关注度低，威慑力不足
  • 我的思考：
    • 结果证明小企业更需关注。它们可能是监管盲区，也是财务风险高发区。那么对于中小企业的关键审计事项披露，应该相较于大规模企业，需要更简明、更有针对性。

📄 《关键审计事项披露能否降低上市公司的股权融资成本?》(审计研究, 2020)

  • 2024年11月
  • 作者：赵玉洁, 万贻健, 方瑄
  • 核心问题：融资成本是企业生存与发展的“生命线”。在信息不对称的市场中，投资者因看不清风险，往往要求更高的回报，推高企业股权融资成本。而披露关键审计事项能否降低上市公司的股权融资成本？
  • 主要发现：
    • 研究发现，企业披露KAMs后，其股权融资成本显著下降；而且披露的事项越多，融资越便宜。这说明投资者把KAMs看作“有用信号”——审计师公开点出高风险领域，反而增强了财报可信度，这是因为信息更透明、会计质量更高、治理更完善。
    • 融资困难的企业（融资约束强）：原本难融资，KAMs成了“信用背书”，更容易获得资金；非“四大”事务所审计、分析师关注少的企业：原本信息透明度低，KAMs填补了信息空白；外部治理差的企业（如媒体关注度低、机构持股少）：KAMs起到了“替代性监督”作用，弥补了外部监督的不足。
  • 我的思考：
    • 关键审计事项不只是合规文件，更是企业“信用资产”的一部分。当审计师愿意公开谈论风险，反而提升了市场信任，帮助企业降低融资成本——这是一种“以透明换信任，以披露换资本”的正向循环。同时，关键审计事项实质上是审计师在资本市场上为企业“作证”。这要求审计师不仅专业过硬，还要具备风险沟通能力。
    • 但是当如果所有公司都披露5–6个KAMs，且内容雷同（如“收入确认”“商誉减值”），投资者可能麻木，反而削弱其信息价值。未来应鼓励差异化、个性化、实质性的披露

📄 《非正式审计团队规模与关键审计事项披露》(会计研究, 2022)

  • 2024年11月
  • 作者：陈丽红, 周佳, 张龙平
  • 核心问题：审计质量不仅取决于正式制度，也深受事务所内部“非正式组织”的影响。在大型会计师事务所中，常存在以合伙人为核心的“非正式审计团队”（即实际执行项目的稳定协作小组），这些团队虽非官方建制，却掌握着项目资源与决策权。那么非正式审计团队的“规模”是否会影响关键审计事项的披露质量。进一步地，团队越大，是更有资源保障审计深度，还是更容易形成“山头主义”、追求效率而牺牲披露充分性？影响在不同情境下（如独立性、工作负荷）是否有所不同？最终是否会影响企业的盈余质量？
  • 主要发现：
    • 非正式审计团队人数越多，其负责的上市公司披露的关键审计事项数量更少，写的文字更短、更笼统。
    • 大团队更少披露反映企业独特风险的“公司特性事项”，写的文本与其他公司高度相似——像是“复制粘贴”，缺乏定制化分析。
    • 由于关键审计事项披露不充分，未能有效揭示财务风险，导致企业后续的会计信息质量变差，利润更易被操纵或失真
  • 我的思考：
    • 某些会计事务所只是形式上的“大所”，但其工作未必有大所应该有的实质的“高质量”。许多事务所实行“分灶吃饭”模式——各非正式团队独立核算、自负盈亏。这导致：团队规模扩张是为了多接项目、多赚钱，而非提升专业能力；为控制成本、赶工期，牺牲审计程序深度；形成“圈子文化”，缺乏统一质量复核，最终导致KAMs沦为合规形式。
    • 对于团队管理而言，应该是推动“团队专业化”而非“团队规模化”。

📄 《关键审计事项段能够提高审计报告的沟通价值吗?》(会计研究, 2018)

  • 2024年12月
  • 作者：王艳艳, 许锐, 王成龙
  • 核心问题：2016年起，中国开始试点在审计报告中披露“关键审计事项”（KAMs），旨在提升审计透明度。但一个核心问题始终存在，即这些新增内容是“真有用”，还是“走过场”？
  • 主要发现：
    • 披露关键审计事项后，市场是真的对其有反应。具体而言，在披露KAMs前后，相关公司的股价反应更强烈，累计超额收益率显著高于未披露公司。这说明：投资者认为这些信息有价值，审计报告不再是千篇一律的“标准模板”，而是传递了新的、影响决策的信息。
    • 写得越具体、信息量越大的KAMs，市场反应越好，并且如果明确指出某事项的责任主体（如“管理层在收入确认中采用了重大判断”），市场更能理解风险归属，反应更积极。
  • 我的思考：
    • 市场是愿意为“更透明的审计”买单。投资者对于审计报告并不是采取一个忽视的态度，而是过去模板化的审计报告内容确实是“没什么可看的”。一旦审计师利用关键审计事项开始说讲“重点”、划“责任”，市场就会用股价投票。同时，这也提示审计师不仅是会计专家，更重要地是，还要成为风险沟通者。

📄 《审计师披露关键审计事项影响公司的审计聘用制度吗》(南开管理评论, 2023)

  • 2024年12月
  • 作者：张睿, 冯均科
  • 核心问题：A+H股上市公司长期以来存在一种制度安排：双重审计——即同时聘请内地和香港的会计师事务所分别出具审计报告。这一做法初衷是增强跨境投资者信心，但成本高昂，且随着两地会计与审计准则趋同，其必要性受到质疑。那么在内地审计师开始披露“关键审计事项”后，企业是否更愿意放弃双重审计，转而由内地事务所“独立担纲”？
  • 主要发现：
    • 研究发现，自从内地审计师开始在报告中披露关键审计事项后，A+H股公司放弃双重审计、选择单一审计（仅用内地所）的概率显著上升
    • 这种效应在小规模内地会计师事务所中尤为明显。原因可能是：大所本身已有品牌背书，而小所长期被质疑专业能力，KAMs的高质量披露成为它们“证明自己”的机会，帮助其赢得客户信任。
  • 我的思考：
    • 信任不是靠“双倍成本”建立的，而是靠“高质量沟通”赢得的。过去，A+H股公司选择双重审计，本质是对内地审计独立性与专业性的“不放心”。而关键审计事项改革，恰是内地事务所的一次“自我证明”的机会——通过主动披露、深度沟通，去获得公司的信任。

2. 机器学习在公司治理中的应用探索

📄 《我国科技企业融资的决定因素研究———基于科创板企业的机器学习分析》(金融研究, 2022)

  • 2025年01月
  • 作者：陆瑶, 施函青
  • 核心问题：2019年科创板开板，标志着中国资本市场向“硬科技”和“创新驱动”转型。其上市审核更强调“科技含量”而非传统盈利指标。那么什么样的科技企业更容易成功登陆科创板？进一步地，是“现在赚不赚钱”重要，还是“未来有没有潜力”更重要？成功上市的企业，是否一定能在二级市场持续获得投资者青睐？
  • 主要发现：
    • 决定企业能否登陆科创板的三大关键因素是：研发水平（有没有核心技术）成长性（收入、用户、市场扩张快不快） 公司治理水平（管理是否规范透明）其中，研发能力最重要，而是否盈利几乎最不重要。也就是未来更重要。
    • 投资者和审核机构更看重企业是否拥有稳定、高比例的科研团队，这是“真搞技术”的标志。
    • 能帮助“成功上市”的因素，不一定能让股票在上市后涨得好；企业如果只盯着怎么“合规过会”，可能忽视真正影响股价的市场因素（如流动性、行业周期、投资者情绪）。
  • 我的思考：
    • 科创板审核背后的反应出，不再是传统IPO的看利润，而更像是一个“潜力识别器”。去欢迎那些“现在不赚钱但未来可能改变世界”的企业。比如，生物医药、芯片、人工智能等长周期行业。
    • 也会存在许多企业拼尽全力只为“过会”，但一旦上市，市场可能“见光死”。因为投资者关注的是持续创新能力、商业化落地能力、现金流健康度，而不仅是审核所需的“技术先进性证明”。

    • 市场常将“卖出评级”视为负面情绪，但本研究表明它是治理工具。监管应鼓励多元化评级分布，避免“清一色增持”的虚假繁荣。

📄 《The Earnings-Return Relation》(The Economics of Accounting)

  • 2025年03月
  • 作者：陈友余, 赵金晶, 王欣, 刘纯霞
  • 核心问题：会计盈余与股票收益之间是什么关系？为什么这种关系是“不对称”的？
  • 主要发现：
    • 会计和股价的关系，并不是“好消息坏消息一视同仁” 。当公司出问题、股价大跌时，财报会很快把亏损记下来；但当公司表现好、股价大涨时，利润却不会立刻全部体现在账上。也就是“坏消息反应快，好消息反应慢”。
    • 特定公司下：刚起步或高速发展的公司，未来潜力大，即使现在亏钱（现金流为负），市场也可能看好它、股价上涨。但一旦环境变差，它们融资困难，就必须靠实实在在的现金流生存，所以亏损时财报反应特别快。而成熟时期的公司赚钱稳定，无论好坏时期都靠现有资产产生现金，所以盈余和股价的关系比较均衡。
  • 我的思考：
    • 会计信息的特性（如反应速度）其实深受企业自身发展阶段和外部经济环境的影响。也就是会计是在“如实反映”企业的脆弱与韧性。
    • 投资者不能只看利润增减，还需要观察这家公司处在什么阶段？它是因为行业下行被迫亏损，还是这家公司像通过烧钱的方式换增长？如果是后者，短期亏损未必是坏事；如果是前者，则要警惕财报突然暴雷的风险。

📄 《中国混合所有制改革与微观企业经济后果研究:进展、综述与展望》(会计研究, 2023)

  • 2025年03月
  • 作者：曹瑾, 张东旭, 杨蓉
  • 核心问题：中国推动国有企业引入非国有资本、非国有企业引入国有资本的“混合所有制改革”，是否真的能提升企业效率、激发活力？其背后的机制是什么？
  • 主要发现：
    • 把国企和民企的股份掺在一起，并不能自动带来好效果。关键是要看“怎么混”，也就是混改过程中是真给民企话语权，还是只是走形式。
    • 国企分三类（商业类国企、公益类国企、战略类国企），混改方式应不同
    • 成功靠两种效应：治理 + 资源；具体而言，民企进来后，如果能派董事、参与决策，就能监督国企中存在的弊端，减少官僚作风，提升效率。另外，双方只有“取长补短”，才能共赢：民企带来市场机制、管理经验；国企给民企带来政策支持、贷款便利。
    • 用人机制上可以通过市场化选聘职业经理，提高企业的治理效率。
  • 我的思考：
    • 混改的真正目标，不是简单地让国企变“民”或民企变“红”，而是通过股权多元化，倒逼企业建立市场化经营机制。不应该只是换汤不换药。
    • 混改要设计好权力制衡机制。不能让改革成为民企为了“贴金”拿资源。

📄 《新技术、企业组织变革与财务管理》(财务研究, 2020)

  • 2025年07月
  • 作者：谢志华, 李莎, 沈彦波
  • 核心问题：在“大智移云物”（大数据、人工智能、移动互联网、云计算、物联网）和区块链等新技术深刻改变商业环境的背景下，传统的、基于实体企业的财务管理理论与实践是否还能适用？如果不能，财务管理应当如何重新定位？
  • 主要发现：
    • 现在的新技术让企业不再是过去的单打独斗，企业不再只靠自己干活，而是通过平台连接供应商、客户、物流、服务商等，大家一起协作完成任务。
    • 价值不是一家公司单独创造的，而是大家一起协作创造的。因为价值产生方式的改变，财务管理也转变为变成“搭平台、促合作、分蛋糕”的角色。
    • 现在技术可以让财务看到每一笔交易、每一个客户行为的实时数据，还能把业务数据（比如订单、库存、物流）和财务数据（收入、成本）打通。这需要财务人员有能够实时掌握全局，快速决策的能力。
  • 我的思考：
    • 未来的财务人员不应该只是自己关在办公室里做报表。必须了解业务、懂技术、会协调外部伙伴。财务的价值在当今的大环境下应该是“创造协同价值”。
    • “价值共享化”就需要设计合理的分成机制、激励机制，确保合作各方“有活干、有钱赚”，未来的财务应该成为连接、协同与共享的重要力量。
