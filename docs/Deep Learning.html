本板块展示了我在以下方向的部分实践成果：

· KAN 网络探索

基于模拟数据构建分类任务，对结合知识图谱与深度学习的 KAN 网络进行实验，评估其在处理复杂关系和多模态信息方面的表现，探索模型在结构化知识与语义学习上的协同能力。

· 表文融合：关键审计事项识别

面向审计文本分析场景，构建“文本+表格”融合框架，将关键审计事项（KAM）描述转化为语义张量，结合深度神经网络实现特征提取与模式识别。该方法提升了对非结构化审计信息的理解能力，增强了模型对风险信号的捕捉精度与可解释性。

 

1.kan网络

本项目通过构建模拟数据对 KAN 网络展开系统性实验，重点评估其在分类任务中的性能表现。作为一种融合知识图谱与深度学习的新型架构，KAN 网络在处理复杂语义关系与多模态信息方面展现出显著潜力，本实验旨在探索其在结构化语义理解与多源信息协同建模方面的能力。

采用模拟数据的优势在于其高度可控性与灵活性。通过预设生成规则与参数配置，能够构建具备清晰特征结构和标签分布的数据集，从而系统评估模型在不同场景下的行为特性与泛化能力。同时，模拟数据有效规避了真实数据中可能存在的隐私与安全问题，使实验流程更加高效、可复现，为模型机理的深入解析提供了良好基础。

📄相关报告详见：深度学习_kan分类（模拟数据）.pdf

2.表文融合

本项目聚焦于关键审计事项的智能识别与分析，提出一种创新的“表文融合”深度学习框架。通过将审计报告中的文本描述与相关表格数据进行统一表征，生成语义张量，并借助深度神经网络实现端到端的特征提取与模式识别，从而提升对关键审计事项的精准识别能力。

该方法不仅加强了对非结构化审计文本的语义理解，也提升了模型对风险信号与审计逻辑的感知精度。
